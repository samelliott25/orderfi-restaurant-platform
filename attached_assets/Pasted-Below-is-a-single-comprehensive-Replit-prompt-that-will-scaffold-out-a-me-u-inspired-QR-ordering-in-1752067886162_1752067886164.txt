   - `/scan` — QR scanner + “Continue as guest”
   - `/menu` — main menu browsing
   - `/cart` — cart drawer slide-out
   - `/status` — order status

3. **Tailwind config** (`tailwind.config.js`) enabling dark mode and your brand gradient (orange→pink).

4. **Core components** under `/src/components/`:
   - `Header.jsx`  
     • Shows venue name/table, search input, cart icon badge.  
   - `CategoryTabs.jsx`  
     • Horizontally scrollable, icon+label tabs (e.g. Starters, Mains, Drinks).  
     • Uses HeadlessUI’s `<Tab>` for ARIA and keyboard nav.
   - `MenuGrid.jsx`  
     • Responsive 2-4-column grid of `ItemCard` components.
   - `ItemCard.jsx`  
     • Card with image, name, price, “+” button. Hover reveals quick-info.  
   - `ItemModal.jsx`  
     • Opens when “+” clicked, shows full description, list of modifier checkboxes with live price update, “Add to Cart” button.
   - `CartDrawer.jsx`  
     • Slide-in from right; lists cart items, qty controls, subtotal, tip input, “Checkout” button.
   - `CheckoutForm.jsx`  
     • Stripe Elements card input; on submit calls backend endpoint and navigates to `/status`.
   - `StatusTracker.jsx`  
     • Polls order status and displays a 4-step timeline.

   - `CartPage.jsx`  
     • Renders `<CartDrawer>`.
   - `CheckoutPage.jsx`  
     • Renders `<Header>` + `<CheckoutForm>`.
   - `StatusPage.jsx`  
     • Renders `<Header>` + `<StatusTracker>`.
