Here’s a step-by-step recipe to drop that exact “molten-glass” orb into a React-based Replit. It’ll give you the same 3D depth, noise, fresnel highlights and bloom you saw in your screenshots.

1. Create a new Replit
Choose the “React.js” template.

Wait for it to spin up.

2. Install the extras
Open the Shell tab at the bottom and run:

npm install three @react-three/fiber @react-three/postprocessing
3. Add your orb component
Create a new file src/OrbCanvas.jsx and paste:

jsx
Copy
Edit
// src/OrbCanvas.jsx
import React, { useRef } from 'react';
import { Canvas, useFrame } from '@react-three/fiber';
import { UnrealBloom } from '@react-three/postprocessing';
import * as THREE from 'three';

const OrbMaterial = {
  uniforms: {
    time:   { value: 0 },
    color:  { value: new THREE.Color(0xff5500) }, // tweak your base hue here
  },
  vertexShader: `
    varying vec3 vNormal;
    void main(){
      vNormal = normal;
      gl_Position = projectionMatrix * modelViewMatrix * vec4(position,1.0);
    }
  `,
  fragmentShader: `
    uniform float time;
    uniform vec3 color;
    varying vec3 vNormal;

    // very simple noise
    float noise(vec3 p) {
      return fract(sin(dot(p, vec3(12.989,78.233,54.53))) * 43758.5453);
    }

    void main(){
      float n = noise(vNormal * 5.0 + time * 0.6);
      float fresnel = pow(1.0 - dot(normalize(vNormal), vec3(0,0,1)), 3.0);
      vec3 col = mix(color * 0.3, color, n) + fresnel * 0.8;
      gl_FragColor = vec4(col, 1.0);
    }
  `
};

function Orb() {
  const mat = useRef();
  useFrame(({ clock }) => {
    mat.current.uniforms.time.value = clock.getElapsedTime();
  });
  return (
    <mesh>
      <sphereGeometry args={[1, 128, 128]} />
      <shaderMaterial ref={mat} {...OrbMaterial} />
    </mesh>
  );
}

export default function OrbCanvas() {
  return (
    <Canvas gl={{ antialias: true }} camera={{ position: [0, 0, 3] }}>
      <ambientLight intensity={0.2} />
      <pointLight position={[5, 5, 5]} intensity={1} />
      <Orb />
      <UnrealBloom luminanceThreshold={0.2} luminanceSmoothing={0.9} height={300} />
    </Canvas>
  );
}

Replace your src/App.jsx with:

import React from 'react';
import OrbCanvas from './OrbCanvas';
import './App.css';

function App() {
  return (

6. Run and tweak
Click “Run” at the top of Replit. You should see a perfectly round, glowing orb with swirling color and bloom—just like your screenshots.

From here you can:

Swap the color uniform to fine-tune your gradient (try 0x9933ff for more purple, or feed it from React props).

Adjust the noise scale (the *5.0 in the fragment shader) for tighter or looser swirls.

Tweak bloom settings to dial in more or less glow.
