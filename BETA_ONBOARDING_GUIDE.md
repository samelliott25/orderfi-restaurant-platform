## Detailed Setup Instructions

### Restaurant Profile Configuration

**Basic Information Required:**
```
Restaurant Name: [Your Restaurant Name]
Cuisine Type: [American, Italian, Mexican, Asian, etc.]
Operating Hours: [Monday-Sunday with specific times]
Address: [Full address for delivery/pickup]
Phone: [Primary contact number]
Email: [Order notifications and support]
```

**Advanced Settings:**
```
Average Preparation Time: [15-45 minutes typical]
Delivery Radius: [Miles/Kilometers if applicable]
Minimum Order Amount: [Optional threshold]
Peak Hours: [Times when orders take longer]
Special Dietary Options: [Vegan, Gluten-Free, etc.]
```

### Menu Item Setup

**Required Fields Per Item:**
```
Item Name: [Clear, descriptive name]
Description: [2-3 sentences highlighting key ingredients]
Price: [In local currency]
Category: [Appetizers, Mains, Desserts, Beverages]
Preparation Time: [Minutes required]
Dietary Tags: [Vegan, GF, Spicy, etc.]
Availability: [Always, Weekends Only, Limited Time]
```

**AI-Friendly Menu Tips:**
- Use descriptive names that customers naturally say
- Include popular ingredients in descriptions
- Mention cooking methods (grilled, fried, baked)
- Highlight signature items or house specialties
- Keep descriptions conversational, not overly technical

### Kitchen Printing Setup

**Supported Printer Models:**
- Epson TM series (TM-T20II, TM-T82II, TM-T88V)
- Star Micronics (TSP143III, TSP654II)
- Bixolon (SRP-350III, SRP-275III)

**Connection Options:**
1. **Ethereum Network** (Recommended): Connect via WiFi, automatic setup
2. **USB Connection**: Direct connection to POS computer
3. **Cloud Printing**: Orders sent via email/cloud service

**Printer Configuration:**
```
Paper Width: 80mm (standard thermal)
Print Quality: Medium (for speed)
Auto-Cut: Enabled
Order Layout: Item name, quantity, special instructions
Footer: Order ID, timestamp, customer info
```

### QR Code Menu Generation

OrderFi automatically generates QR codes for each restaurant:

**QR Code Types Available:**
1. **Table Ordering**: Direct link to AI chat interface
2. **Menu Browser**: Traditional menu view with AI chat option
3. **Rewards Signup**: Customer loyalty program enrollment
4. **Feedback**: Post-meal review and rating system

**QR Code Customization:**
- Restaurant logo integration
- Brand color matching
- Multiple sizes (table tents, window decals, business cards)
- Trackable analytics for scan-to-order conversion

### AI Conversation Training

**Common Customer Requests to Prepare For:**
```
"What's good here?" → Highlight 2-3 signature dishes
"Do you have vegetarian options?" → List plant-based items
"How spicy is the [item]?" → Describe spice level clearly
"Can I customize [item]?" → Explain available modifications
"How long will it take?" → Provide realistic timing
"Do you deliver?" → Confirm service area and minimums
```

**AI Response Customization:**
```javascript
// Example restaurant personality settings
tone: "friendly and professional"
specialties: ["craft burgers", "local ingredients", "gluten-free options"]
signature_items: ["Truffle Burger", "Sweet Potato Fries"]
upselling: "moderate" // suggest sides, drinks, desserts
local_references: ["downtown location", "family-owned since 2015"]
```
