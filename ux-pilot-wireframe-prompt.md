Product Requirement: OrderFi AI - Complete Platform Wireframes

Purpose: To create a revolutionary blockchain-first AI restaurant platform that transforms dining experiences through intelligent conversational interactions. The platform serves two distinct user groups: customers using AI-powered ordering and restaurant staff managing operations through an AI-assisted dashboard.

UI Components:

Customer Interface (OrderFi):
- Home Screen: Video logo animation with gradient "Enter DApp" button, loading states with orange animations
- AI Chat Interface: Full-screen conversational interface with bottom input panel, voice wave animations, quick action pills
- Voice Input: Speech-to-text with visual feedback, pulsing microphone button, real-time transcription display
- Menu Discovery: AI-mentioned items appear as inline clickable buttons within chat messages, expandable item details
- Cart Management: Items added through conversation flow, floating cart indicator, AI-guided checkout process
- Order Tracking: Real-time status updates, progress indicators, blockchain transaction confirmations
- Token Rewards: Multi-tier loyalty display (Bronze/Silver/Gold/Platinum), point balance, redemption interface

Restaurant Dashboard:
- Navigation Bar: Global navigation with order management, kitchen printing, token rewards, settings
- Dashboard Overview: Live statistics widgets, real-time order queue, kitchen display integration
- Order Management: Status cards with drag-and-drop functionality, batch operations, filtering options
- Kitchen Printing: Printer discovery interface, driver installation workflows, print queue management
- AI Operations Chat: Bottom chat interface for staff assistance, similar to customer interface but staff-focused
- Settings Panel: Tabbed interface for system configuration, printer setup, notification preferences

Visual Style:
- Theme: Light theme with #FCFCFC background throughout, optional dark mode for dashboard
- Primary color: Orange #FF6B35 for CTAs, branding, and active states
- Secondary color: Charcoal #1A1A1A for primary text and headers
- Accent color: Gray #6B7280 for secondary text and icons
- Success: Green #10B981 for positive states and confirmations
- Warning: Amber #F59E0B for alerts and pending states
- Error: Red #EF4444 for errors and failures
- Spacing: Consistent 20px outer padding on mobile, 16px between chat messages, 24px section spacing
- Borders: 1px solid #E5E7EB on cards and inputs; 12px radius on chat bubbles, 8px on buttons and cards
- Typography: "Playwrite Australia Victoria" handwritten font for OrderFi headings and branding, system sans-serif medium weight (500) for UI text, regular (400) for body content, base size 16px on mobile
- Icons/images: Modern diamond/sparkle AI icons, filled vector icons for actions, voice wave animations for speech input
